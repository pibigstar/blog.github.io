title: Mysql中的SQL函数
date: 2018-09-20 22:10:31
tags: [mysql,sql]
categories: 数据库

算术函数

  • ABS ：取绝对值
  • MOD：取余
  • ROUND：四舍五入时指定小数点位数

字符串函数

  • CONCAT： 将多个字符串连接起来
  • LENGTH： 计算字段的长度，一个汉字算三个字符，字母和数字算一个字符
  • CHAR_LENGTH：所有都算做一个字符
  • LOWER：全部转为小写
  • UPPER：全部转为大写
  • REPLACE：替换字符
  • SUBSTRING：截取字符串（字符串, 开始截取位置, 截取长度）

日期函数

  • CURRENT_DATE：系统当前日期
  • CURRENT_TIME：系统当前时间
  • CURRENT_TIMESTAMP：系统当前时间（日期+时间）
  • ECTRACT：抽取具体的年、月、日
  • DATE：返回时间的日期部分
  • YEAR：返回时间的年份部分
  • MONTH：返回时间的月份部分
  • DAY：返回时间的天数部分
  • HOUR：返回时间的小时部分
  • MINUTE：返回时间的分钟部分
  • SECOND： 返回时间的秒部分

转换函数

  • CAST：强制数据类型转换
  • COALESCE：返回第一个非空数值

聚合函数

  • SUM：求和
  • AVG：求平均值
  • MAX：找出最大值
  • MIN：找出最小值
  • COUNT：计数
