微信支付痛点

对于大多数同学来说，要开发微信支付可不简单。
附上微信支付官方文档https://pay.weixin.qq.com/wiki/doc/api/index.html
从文档上可以看出，你需要解决很多问题，我就随便挑几个吧。

xml与对象的互转: 微信使用xml格式而不使用json通信, 也确实有点奇葩
签名:千万不要以为只是MD5一下，要是你自己开发，我敢说，你60%的时间都耗在签名调试这一块
